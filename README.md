## 📄 Sobre o Projeto

Este projeto consiste em uma **Landing Page de Alta Conversão** desenvolvida para o escritório de advocacia **A. Melo & M. Gomes**, localizado em Curitiba/PR.

O objetivo foi criar uma identidade digital séria, elegante e funcional, permitindo que clientes conheçam as áreas de atuação (Família, Previdenciário, Trabalhista) e entrem em contato rapidamente.

---

## ✨ Funcionalidades

O site foi desenvolvido com foco na experiência do usuário (UX) e conversão:

- **🎨 Design Responsivo:** Adapta-se perfeitamente a celulares, tablets e computadores.
- **👁️ Efeito Parallax:** Profundidade visual na imagem de capa para um visual moderno.
- **💬 Botão WhatsApp Pulsante:** Call-to-Action (CTA) animado para incentivar o contato imediato.
- **📍 Mapa Inteligente:** Integração com Google Maps e botão de "Traçar Rota" via GPS/Waze.
- **🔒 Compliance LGPD:** Banner de consentimento de cookies funcional.
- **⚡ Performance:** Código otimizado (Vanilla JS) sem dependências pesadas, garantindo carregamento rápido.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais (Puro "Vanilla"):

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização avançada (Flexbox, Grid Layout, Animações Keyframes e Variáveis CSS).
*   **JavaScript (ES6):** Manipulação do DOM, Scroll Suave, Intersection Observer (animações ao rolar) e lógica do Cookie Banner.
*   **FontAwesome:** Ícones vetoriais.
*   **Google Fonts:** Tipografias *Playfair Display* (Títulos) e *Lato* (Textos).
